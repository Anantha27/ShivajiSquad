State Portal Services Page

How To Apply Services

To apply for online citizen services, citizen must login to the State Portal. Please create a new user account, if not registered. 

 

 To Create a new Account

    1) Click on SignIn 

    2) Click on Create Account 

    3) Register with the state portal.

    4) Login to the portal and

    5) Avail citizen services from Apply Online Tab.
